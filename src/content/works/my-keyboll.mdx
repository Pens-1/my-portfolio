## Overview

自作の無線トラックボール内蔵キーボード。本体ハードウェアもさることながら、この作品の特徴は **KiCad の回路図(.kicad_sch)と基板(.kicad_pcb)を Python から生成している** こと。

## Generator pipeline

- `gen_schematic.py`: ピン割当てテーブルから `.kicad_sch` を組み立てる
- `gen_pcb.py`: 部品配置と基本的な配線を `.kicad_pcb` に出力する
- `gen_schematic_keyball.py`: Keyball 系の派生版を自動生成

## Hardware

- MCU: SuperMini nRF52840
- Optical sensor: PMW3610 (3.3V 直動作、LDO 不要、SPI接続)
- Battery: JST-PH 2ピン、BATIN (P0.04) 経由

## Pin mapping (excerpt)

```
MOSI  : P0.10  → PMW3610 pin6
MISO  : P1.11  → PMW3610 pin7
SCK   : P1.13  → PMW3610 pin5
NCS   : P1.06  → PMW3610 pin8
MOTION: P1.04  → PMW3610 pin4
```

## Why generate from Python

- ピン割当ての変更 → 回路図の再描画が一発
- 派生版(Keyball 39 / 自作レイアウト) を同じコードベースで維持
- DRC(デザインルールチェック)結果も自動生成ログで追える
