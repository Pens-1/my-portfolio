## Overview

**my-keyboll** は自作の無線トラックボール内蔵キーボード。そしてこの作品の真の特徴は、**KiCad の回路図(.kicad_sch)と基板(.kicad_pcb)を Python から生成している** こと。さらにその自動化を一段メタ化した **mcp-kicad-sch-api**(AI agent が KiCad を操作できる MCP サーバー)までセットで開発した。

## Generator pipeline

- `gen_schematic.py` — ピン割当てテーブルから `.kicad_sch` を自動生成
- `gen_pcb.py` — 部品配置と配線を `.kicad_pcb` に出力
- `gen_schematic_keyball.py` — Keyball 系派生版を同じコードベースで維持
- `make-keybord/` — 3代目の PCB generator(ライブラリ化途中)

## Hardware

- MCU: SuperMini nRF52840
- Optical sensor: PMW3610 (3.3V 直動作、LDO 不要、SPI接続)
- Battery: JST-PH 2ピン、BATIN (P0.04) 経由
- Firmware: ZMK(BLE + 省電力)

## mcp-kicad-sch-api

別リポジトリで開発している MCP サーバー。**AI エージェントから KiCad の schematic を操作できる** API を提供する:

- コンポーネント追加・削除・配線
- 部品ピン位置の取得、ラベル付与
- 図の検証(DRC 実行)
- 階層シートの操作

これで「Python スクリプトで PCB を生成する」自動化を、さらに「自然言語で AI エージェントに PCB を作らせる」レイヤーまで押し上げた。

## Why this matters

電子回路設計(高専・専門領域)× Python 自動化 × MCP(LLM エコシステム)を縦に貫いているのがポイント。**ハードとソフトの境界を自分で設計できる** ことを示すための1本。
