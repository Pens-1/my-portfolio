# Project Overview

Pythonを使用し、複雑なブラウザ操作を自動化するシステムです。
PlaywrightとSeleniumを使い分け、対スクレイピング対策が施されたサイトからも安定してデータ収集を行えます。

## Key Features

- **ヘッドレスブラウザ** による高速な処理
- **AI (LLM)** を活用した動的なコンテンツ解析
- **Docker** コンテナによるポータブルな実行環境
