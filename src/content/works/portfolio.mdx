# Project Overview

このポートフォリオサイト自体も、私の技術力とデザインセンスを表現する重要な作品です。
「洗練されたプロフェッショナル」をテーマに、黒と金を基調としたデザインを採用しました。

## Technical Highlights

### 🎨 Modern UI/UX
- **Tailwind CSS** による効率的かつ柔軟なスタイリング
- **Intersection Observer** を使用したスクロールアニメーション
- **CSS Modules** や **BEM** ではなく、Utility-firstなアプローチを採用

### 🏗 Architecture
- **Vite** による高速な開発体験とビルド
- **MDX** によるコンテンツ管理（現在移行中）
- **Docker** による一貫した開発環境

## Design Philosophy

> "Simple is consistent."

余計な装飾を削ぎ落とし、コンテンツ（実績）が最も映えるようなミニマルなデザインを心がけました。
