# Project Overview

カメラ映像やセンサーを使用して、部屋の滞在人数をリアルタイムにカウント・可視化するシステムです。
空き状況を外部から確認できるようにし、利用効率を向上させます。

## Key Features

- リアルタイムな人数カウント
- 混雑状況のヒートマップ表示
- プライバシーに配慮したデータ処理
