# Project Overview

大規模な部活動の運営を効率化するための管理システムです。
練習メニューの作成、出席管理、進捗状況の可視化を一元管理します。

## Key Features

- 練習スケジュールのドラッグ＆ドロップ編集
- メンバーごとの出席率・成長記録の可視化
- LINE Bot と連携した通知機能
