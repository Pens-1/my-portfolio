## Overview

Ollama で動くローカルモデル(Qwen3 / Gemma3 / mistral など)を LLM バックエンドに据え、ブラウザを自動操作するエージェント。**クラウド API を一切呼ばない** のが設計の軸。

## Two modes

1. **Function Calling 1段階版(推奨)**
   - 思考と行動を1回のLLM呼び出しで完結
   - 構造化出力で安定、レイテンシ半減

2. **2段階版(Thought → Action)**
   - 思考フェーズと行動フェーズを明示的に分離
   - 複雑タスクでのデバッグ性が上がる

## Config snapshot

```python
AGENT_MODEL    = "gemma3:12b"
OLLAMA_NUM_CTX = 13000
HEADLESS_MODE  = False
```

## Why local

- クラウド API コストがかからない
- 社内データや個人データを外部に出さずに試行できる
- モデル差し替え(Qwen ↔ Gemma ↔ Mistral)で挙動が比較できる
